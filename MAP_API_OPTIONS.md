# 무료 지도 API 옵션

## 추천 순위

### 1. **OpenStreetMap + Leaflet** ⭐ (가장 추천)
- **완전 무료**: 무제한 사용 가능
- **오픈소스**: 커뮤니티 기반
- **한국 지원**: 한국 지도 타일 제공
- **가벼움**: 빠른 로딩 속도
- **설정 간단**: API 키 불필요

**장점:**
- 완전 무료, 제한 없음
- 한국 지도 타일 제공 (다음 지도, 카카오맵 타일 사용 가능)
- 가벼운 라이브러리
- 커스터마이징 용이

**단점:**
- 네이버 지도만큼 상세하지 않을 수 있음
- 한국어 주소 검색은 별도 서비스 필요

### 2. **Google Maps API**
- **무료 티어**: 월 10,000회 요청 무료 (2025년 기준)
- **한국 지원**: 완벽한 한국 지도 및 주소 검색
- **API 키 필요**: Google Cloud 계정 필요

**장점:**
- 한국 지도 및 주소 검색 완벽 지원
- 상세한 지도 정보
- Geocoding API 제공

**단점:**
- 무료 티어 제한 (월 10,000회)
- 초과 시 요금 발생
- API 키 설정 필요

### 3. **MapLibre GL JS**
- **완전 무료**: 오픈소스
- **고성능**: 벡터 타일 기반
- **커스터마이징**: 스타일 자유롭게 변경 가능

**장점:**
- 완전 무료
- 고성능 렌더링
- 아름다운 스타일

**단점:**
- 한국 타일 소스 별도 설정 필요
- 초기 설정이 다소 복잡

### 4. **카카오맵 API**
- **무료 티어**: 월 300,000회 무료
- **한국 최적화**: 한국 지도 및 주소 검색 완벽
- **API 키 필요**: 카카오 개발자 계정 필요

**장점:**
- 한국 지도 및 주소 검색 완벽
- 넉넉한 무료 티어
- 한국어 문서 제공

**단점:**
- 무료 티어 제한 (월 300,000회)
- 초과 시 요금 발생
